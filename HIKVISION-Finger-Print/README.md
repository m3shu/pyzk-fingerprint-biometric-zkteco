# Device Setup & Configuration Procedure

1. Reset the Device

	Perform a factory reset of the device to ensure a clean configuration environment.

2. Install and Configure via SADP Tool

	Download and install the SADP Tool from the official Hikvision website:
	https://www.hikvision.com/en/support/tools/hitools/clea8b3e4ea7da90a9/

	Launch SADP and complete the following steps:

	Activate the device.

	Set a strong Admin password.

	Assign a static IP address.

	Configure the Default Gateway.

3. Access Device via Web Browser

	Open a web browser and enter the device IP address.

	Log in using:

	Username: admin

	Password: (Newly configured admin password)

4. Configure Administrator Profile

	Create or update the Administrator profile.

	Enroll the following biometric data:

	Fingerprint

	Face recognition data

5. Add Additional Users

	Add all additional user profiles via the web interface.

	Enroll fingerprints directly from the device for each user.

6. Wi-Fi Configuration (Optional)

	The device supports Wi-Fi connectivity.

	When enabling Wi-Fi:

	Assign a different IP address from the LAN configuration to avoid IP conflicts.
